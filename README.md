# ALU UVC - Universal Verification Component

## Overview
This project implements a simple **UVM (Universal Verification Methodology) verification component** for an 8-bit Arithmetic Logic Unit (ALU).

## Purpose
The ALU UVC provides a reusable, modular verification framework to test and validate an 8-bit ALU design that supports the following operations:

- **Addition** - Arithmetic addition of two 8-bit operands
- **Subtraction** - Arithmetic subtraction of two 8-bit operands
- **Bitwise AND** - Logical AND operation between two operands
- **Bitwise OR** - Logical OR operation between two operands
- **Bitwise XOR** - Logical XOR operation between two operands
- **NOT** - Bitwise complement of an operand
- **Shift Left** - Arithmetic left shift operation
- **Shift Right** - Arithmetic right shift operation

## Features
- UVM-compliant agent, driver, monitor, and scoreboard
- Comprehensive test cases covering all ALU operations
- Stimulus generation and response checking
- Coverage analysis and reporting
